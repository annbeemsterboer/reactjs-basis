Om een pagina interactief te maken is er een samenspel tussen
HTML en JavaScript nodig. De interactie tussen deze twee gebeurt door middel
van events. Denk bijvoorbeeld aan het klikken op een button (een gebruikersactie)
of het laden van de pagina. Andere voorbeelden zijn op een toets drukken
of het bewegen van de muis.

Events afhandelen in React is vergelijkbaar met het afhandelen van "gewone" JavaScript DOM events.
Er zijn een aantal verschillen, die lees je in de [React docs](https://reactjs.org/docs/handling-events.html).
