Html elementen hebben een aantal native eigenschappen oftewel props. Kijk maar eens naar:

```jsx
<a href='https://website.com' target='blank'>
  Ga naar website.com
</a>
```

In het element zijn de props 'href' en 'target' meegegeven.
Omdat deze props onderdeel zijn van de W3C standaarden worden ze door browsers herkent.
In React mag je voor custom componenten zelf de props bepalen.
Binnen je compont kun je bijvoorbeeld de rendering laten afhangen van de props.

Meer informatgie over props vind je in de [React docs](https://beta.reactjs.org/learn/passing-props-to-a-component).
