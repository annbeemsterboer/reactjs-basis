Het komt regelmatig voor dat je verschillende componten wilt laten reageren op
een event dat plaats heeft gevonden in een ander component. Een van de manieren
om dat te doen is door de state op component niveau naar boven te verplaatsen.
Vervolgens kun je state in de parent door middel van props doorgeven aan child
components.
